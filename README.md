## Getting Started

1. **Clone the repository:**

   ```bash
   git clone https://github.com/eyekavya/PizzaApp-React.git
   ```

2. **Install dependencies::**

   ```bash
   cd PizzaApp-React
   npm install
   ```

3. **Run the app::**

   ```bash
   npm start
   ```

4. **Open in your browser:**

Visit http://localhost:3000 to start customizing your pizza!
